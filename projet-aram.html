ARAM Cup

Site web officiel d'un tournoi communautaire League of Legends en mode ARAM, avec classements et statistiques générés automatiquement via Python et Google Sheets.

Objectifs du projet

L'ARAM Cup est un tournoi communautaire organisé autour du mode ARAM (All Random All Mid) de League of Legends. L'objectif était de créer un site vitrine officiel pour le tournoi, capable d'afficher des classements et statistiques toujours à jour, sans backend complexe ni base de données dédiée.

  • Présenter le tournoi et ses participants dans un style e-sport immersif
  • Afficher les classements individuels et par équipes automatiquement
  • Centraliser les règles, informations et profils des joueurs
  • Automatiser la mise à jour des classements via un bot Discord
  • Rester maintenable par une seule personne sans serveur dédié

Architecture du système

Le projet repose sur une architecture statique augmentée par un pipeline d'automatisation : les données du tournoi vivent dans Google Sheets, un script Python les traite et génère des images de classement, qui sont ensuite directement embarquées dans le HTML.

Fonctionnement du pipeline de données

La force du projet réside dans son pipeline automatisé. Voici le flux complet, de la saisie des données à l'affichage sur le site :

  • L'organisateur saisit les résultats des matchs dans Google Sheets
  • Une commande Discord (!update) déclenche le script Python
  • Le script se connecte à l'API Google Sheets via un Service Account
  • Il récupère et traite les données (kills, victoires, classements)
  • Il génère les images PNG de classement via Pillow (superposition sur template)
  • Les images sont sauvegardées dans le dossier assets/img/rankings/
  • Le site HTML les charge automatiquement — les classements sont à jour

Structure du site ARAM Cup

Le site comprend plusieurs pages interconnectées :

  • Accueil — Présentation du tournoi, bannière immersive, derniers résultats
  • Participants — Galerie de cartes joueurs avec pseudo, rang, réseaux sociaux
  • Classement — Images générées automatiquement (solo + par équipes)
  • Règles — Règlement complet du tournoi, format, fair-play
  • Header / Footer communs — Navigation et informations partagées
